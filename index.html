    • Three reasons for adding static typing to JavaScript

    1. You can avoid mysterious errors like the classic 'undefined' is not a function
    2. It is easier to refactor code without breaking it
    3. Navigating complex large-scale systems is no longer scary

    TypeScript benefits

  • Better integration with text editors
  • Types help self document the code
  • Catches bugs before compilation
  • Can compile down to a target
    ES version
  • Get early access to future
    features of JS
  • It is faster to refactor

    • Reasons why you should choose TypeScript over JavaScript

    1. TypeScript is more reliable
    2. TypeScript is more explicit
    3. TypeScript and JavaScript
      are practically interchangeable,
      so why not?

    TypeScript drawbacks

  • Types are only checked at compile time
  • Unit tests are no longer instant
  • Non-TS libraries require types
  • Does not reduce bug density
  • Compilation time is increased
  • Build pipelines take longer
  • More code needs to be written
